# Rise at Seven - Website Redesign

A modern, recreation of the Rise at Seven agency website, an award-winning search-first content marketing agency.

---

![Page](https://i.ibb.co.com/nNtVcdCZ/Screenshot-2026-05-09-040417.png)

## Live Demo

[https://theriseatseven.vercel.app/](https://theriseatseven.vercel.app/)

### Features

- Fully responsive design optimized for all devices
- Smooth scroll animations and interactions
- Fast page loads with Next.js App Router
- Modern and accessible UI components
- SEO-friendly structure
- Clean and maintainable styling with Tailwind CSS

### Technologies Used

- Next.js 16
- TypeScript
- Tailwind CSS v4
- Framer Motion
- ESLint
